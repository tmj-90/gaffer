# Voice & messaging framework

A brand is verbal as well as visual. Define the voice and the core messages so copy across the
product reads like one author, not a committee.

## Voice vs tone

- **Voice** = the brand's personality. Consistent everywhere.
- **Tone** = how that voice flexes to context. A friendly brand is celebratory in a win,
  empathetic in a support reply — same voice, different tone.

Place the voice on four spectrums and state where it sits and *why*:

```
Formal    ←─────────→ Casual      (legal docs … social)
Simple    ←─────────→ Complex     (consumer … technical B2B)
Serious   ←─────────→ Playful     (finance … entertainment)
Reserved  ←─────────→ Expressive  (corporate … lifestyle)
```

## Define the voice

Pick **3–5 personality traits**, each framed as "X, not Y" to draw the boundary, and give each
a do/don't example:

- Confident, *not* arrogant
- Friendly, *not* unprofessional
- Knowledgeable, *not* condescending
- Innovative, *not* gimmicky

```
Trait: Confident, not arrogant
  ✓ "This is the fastest way to ship."
  ✗ "Obviously you'd be foolish not to use this."
```

## Messaging hierarchy

Derive specific messages from the top down so they all ladder up to one positioning:

```
Mission (why we exist)
  → Vision (where we're going)
    → Value proposition (what we offer)
      → Positioning (how we're different)
        → Key messages (what we repeat)
          → Proof points (why to believe)
```

Fill-in templates:

```
Mission:       We [action] for [audience] by [method] so they can [outcome].
Value prop:    For [customer] who [need], [Brand] is a [category] that [key benefit].
               Unlike [competitor], we [differentiator].
Positioning:   [Brand] is the [category] for [audience] who want [outcome]
               because [reason to believe].
```

## Consistency checklist (audit copy + visuals against this)

**Visual**
- [ ] Only brand palette colours; 60/30/10 ratio roughly held
- [ ] Brand fonts, correct weights, proper hierarchy
- [ ] Logo: approved variant, clear space, no unauthorised modification, legible at size
- [ ] Imagery matches the brand style and quality bar
- [ ] Contrast meets WCAG AA (4.5:1 text / 3:1 large+UI)

**Verbal**
- [ ] Tone matches the defined personality and the context
- [ ] Brand terminology, capitalisation, and abbreviations consistent
- [ ] Copy ladders up to a key message and a proof point
- [ ] No conflicting or off-brand claims

If positioning or audience is genuinely undecided, **name the gap** — don't fabricate a
strategy to look complete. A flagged open question is worth more than a confident guess.
